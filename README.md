# Học tiếng Việt — 越語核心 989

純前端、可離線使用的越南語單字學習網頁:989 個核心單字 + 真人級 TTS 發音(Google Cloud TTS, vi-VN-Wavenet-A)。

**線上使用**:開啟 GitHub Pages 網址即可,無需安裝。學習進度自動存在你自己瀏覽器的 localStorage(別人看不到),可用頁面底部的「匯出/還原」備份或搬移進度。

## 功能

- **學習**:萊特納盒子 + SRS 到期複習(盒1=1天 → 已熟=30天)、收藏(⭐)、只學收藏模式
- **測驗**:看字選義 / 聽音選義 🎧 / 看中文選越文;答錯統計,連錯 2 次自動收藏
- **瀏覽**:全文搜尋、最常錯排序、點單字直接跳學習卡片
- **常用**:15 個分類 334 條(數字到億、動物、顏色、食物、日期星期、方位、家人稱謂、打招呼、購物殺價、餐廳點餐、交通、身體緊急、天氣、量詞、國家地名),全部真人發音
- 例句語音、鍵盤快速鍵(空白翻面、←/→ 作答、F 收藏、1–4 選測驗答案)、連續學習天數 🔥

## 開發

- `index.html` 為唯一前端檔案(單字資料 `WORDS` 與常用詞 `CATS` 內嵌其中)
- `audio/w/` 單字、`audio/sent/` 例句、`audio/c/` 常用詞 mp3
- `scripts/generate-*.mjs`:以 Google Cloud TTS 產音檔(可續跑,需自備 `GOOGLE_APPLICATION_CREDENTIALS` 金鑰,金鑰勿入版控)
- 設計文件見 `docs/superpowers/specs/`
